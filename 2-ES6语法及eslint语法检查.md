#### ES6语法

1. 如果对象的键-值一样，则值可省略
```
{
router:router
}
#可简写成
{
router
}
```
2. // 注释需要在双斜杠后空一格
3. 每个页面代码的最后一行需要空一行
4. data中的空格
```
 data () {
    return {
      swiperOption: {}
    }
  }
```
5.对象中eslint语法要求
对象中eslint语法要求
> key后面紧跟冒号:,value前要有一个空格
```
export default {
  name: 'Home',
  components: {
    HomeHeader
  }
}
```
