##### 编译安装

    ``` bash
    # 安装依赖
    npm install
    
    # 使用热加载方式启动本地服务 localhost:8080
    npm run dev
    
    # 最小化编译生成发布版本
    npm run build
    
    # 最小化编译生成发布版本,并查看分析报告
    npm run build --report
    ```
